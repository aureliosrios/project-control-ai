Soporte Técnico de Obra

"No se cargan las actividades ni el EDT. Si el index requiere un archivo externo CSV, ¿cómo lo soluciono para mis pruebas?"
Solución: Para hacer pruebas rápidas y estables, elimina el vínculo externo al CSV. En su lugar, incrusta la información de prueba de las actividades directamente dentro del objeto JavaScript en tu index.html. Esto evita bloqueos de red y fallos de carga iniciales.
AE

Alumno PCAI

Ingeniero de Planificación

"Cuando desgloso el EDT y deseo elegir la actividad, no hay información de elección. Deseo que incluyas algunas de prueba."
Solución: Se definió una base de datos local llamada baseDatosEDT. Al seleccionar un EDT, se dispara el evento onchange, el cual limpia el selector de actividades e inserta dinámicamente únicamente las tareas correspondientes a ese frente de trabajo (ej. Excavación en cimientos, montaje de columnas).

"¿Cómo conecto mi espacio de trabajo local con el repositorio remoto de GitHub usando PowerShell?"

Google Apps Script

Webhook Receptor con Cabeceras CORS Soportadas

Macro para Google Apps Script que procesa el JSON del formulario y hace append seguro a Google Sheets.

function doPost(e) {
  var corsHeaders = {
    "Access-Control-Allow-Origin": "*",
    "Access-Control-Allow-Methods": "POST, GET, OPTIONS",
    "Access-Control-Allow-Headers": "Content-Type"
  };
  
  if (e === undefined) {
    return ContentService.createTextOutput("OPTIONS OK").setHeaders(corsHeaders);
  }

  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  
  sheet.appendRow([
    new Date(),
    data.supervisor,
    data.edt,
    data.actividad,
    data.horas,
    data.avance,
    data.observaciones
  ]);

  return ContentService.createTextOutput(JSON.stringify({status: "success"}))
                       .setMimeType(ContentService.MimeType.JSON)
                       .setHeaders(corsHeaders);
}

function doOptions(e) {
  return ContentService.createTextOutput("")
                       .setHeaders({
                         "Access-Control-Allow-Origin": "*",
                         "Access-Control-Allow-Methods": "POST, GET, OPTIONS",
                         "Access-Control-Allow-Headers": "Content-Type"
                       });
}
JavaScript / HTML

Modelo de Datos del EDT e Inserción Dinámica

Estructura de datos embebida en JavaScript para poblar los selectores en cascada (EDT ➡️ Actividad) libre de archivos CSV externos.

const baseDatosEDT = {
  "EDT-01: Cimentaciones y Obras Civiles": [
    { codigo: "ACT-101", nombre: "Excavación masiva y perfilado de terreno" },
    { codigo: "ACT-102", nombre: "Habilitado de acero estructural de refuerzo" },
    { codigo: "ACT-103", nombre: "Encofrado y vaciado de concreto premezclado" }
  ],
  "EDT-02: Estructuras Metálicas": [
    { codigo: "ACT-201", nombre: "Montaje mecánico de columnas estructurales" },
    { codigo: "ACT-202", nombre: "Alineamiento y torqueado de pernos" }
  ]
};

function actualizarActividades() {
  const edtSelect = document.getElementById('select-edt');
  const actSelect = document.getElementById('select-actividad');
  const seleccionado = edtSelect.value;

  actSelect.innerHTML = '';
  
  if (seleccionado && baseDatosEDT[seleccionado]) {
    baseDatosEDT[seleccionado].forEach(act => {
      let option = document.createElement('option');
      option.value = act.codigo + " - " + act.nombre;
      option.textContent = act.codigo + " - " + act.nombre;
      actSelect.appendChild(option);
    });
    actSelect.disabled = false;
  } else {
    actSelect.disabled = true;
  }
}
PowerShell / Git

Secuencia de Terminal para Despliegues

Línea secuencial de comandos para inicializar, configurar el repositorio remoto en GitHub y subir el index.html.

# 1. Inicializar repositorio local
git init

# 2. Agregar origen remoto
git remote add origin https://github.com/aureliosrios/Clase_5.git

# 3. Descargar referencias de ramas
git fetch origin

# 4. Crear y apuntar a la rama main
git checkout -b main

# 5. Agregar y confirmar cambios
git add .
git commit -m "feat: implementar index.html con EDT embebido y conexión a Google Sheets"

# 6. Desplegar
git push -u origin main

  • Excelente compatibilidad móvil para reportes directamente en el campo.
