1. Introducción a los Entornos de Agente Locales Concepto base

Se explicó el salto tecnológico de los chats convencionales en la nube a los Entornos de Desarrollo Integrados de Agente (AIDE). A diferencia de las interfaces comunes, un AIDE tiene control directo sobre el sistema de archivos del usuario, permitiendo pair-programming de forma ágil y real.

2. Instalación Guiada de Antigravity IDE Instalación

Paso a paso de la descarga de Antigravity IDE para Windows x64. El docente explicó la diferencia entre la versión CLI (Command Line Interface) y el IDE completo, prefiriendo este último debido al control minucioso de archivos que otorga directamente al programar.

3. Vinculación del Workspace y Cuentas de Google Configuración

Proceso de inicio de sesión y vinculación de cuentas de Google (Gmail) dentro del IDE. Vinculación del primer directorio local de trabajo (Select Folder/Workspace) en el sistema operativo Windows para habilitar las habilidades de lectura y escritura del agente de IA.

Control de Directorio Local

Antigravity IDE opera sobre una carpeta local del sistema. Todo código creado, depurado o modificado se guarda en tiempo real en la computadora del usuario.

La instalación sigue el flujo clásico de Windows (Siguiente, Siguiente, Aceptar). La vinculación requiere otorgar permisos de seguridad del navegador para iniciar la sesión.

smart_toy Antigravity IDE Software Clave

El entorno de desarrollo agéntico de Google. Cuenta con herramientas integradas de pair-programming, acceso a la consola local, capacidades de búsqueda semántica de código y edición de archivos.

terminal Antigravity CLI (Command Line Interface) Línea de Comandos

Versión del agente para terminales de comandos. Recomendado para usuarios avanzados que prefieren interactuar sin interfaz gráfica o automatizar llamadas en segundo plano mediante scripts de shell.

Analiza la estructura del workspace actual y genera un archivo README.md que describa el propósito del proyecto y las dependencias de scripts encontradas.

Escribe un script en Node.js que realice una copia de seguridad en formato zip de la base de datos local y la suba a nuestra carpeta de respaldos.
info Útil para instruir al agente a escribir herramientas de administración de datos directamente en el disco.
