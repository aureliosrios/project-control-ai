construction Configuración del Entorno de Desarrollo

1. Antigravity IDE IDE Agéntico

El entorno de desarrollo local agéntico que se conecta directamente con tu sistema. Permite a la IA leer, buscar y modificar el código del proyecto de forma interactiva y con total trazabilidad.

Ir a Antigravity open_in_new
2. Instalación de Python 3.11 Requisito

Se requiere Python 3.11 para ejecutar los scripts de automatización locales. Se recomienda la instalación a través del gestor de paquetes de Windows Winget para simplificar el proceso.

3. Librerías de Automatización Requeridas Python Packages

Conjunto de paquetes fundamentales para interactuar con datos e interfaces de ingeniería:
• Hojas de Cálculo: openpyxl para generar plantillas de presupuestos.
• Documentos: python-docx para reportes automáticos.
• Planos CAD: pyautocad y ezdxf para interactuar con AutoCAD.
• Documentos PDF: pypdf y pdfplumber para extraer información de metrados y planos.

bolt Comandos Rápidos

Instalar Python

winget install -e --id Python.Python.3.11

Instalar Librerías

pip install openpyxl python-docx pyautocad ezdxf pypdf pdfplumber

labelPrompt: Diseñador de Viga con Acero Comercial y Estribos
Puedes generar una aplicación para diseño de una viga simplemente apoyada en donde el usuario ingrese los datos de: base de la viga, peralte, tipo de concreto, longitud, cargas vivas y muertas, puntuales, y como resultado deseo que la aplicación proporcione: esfuerzos, momentos, cantidad de acero entre otros datos relacionados a este tipo de aplicaciones en HTML. Mejora el plan considerando la normativa peruana, zona sísmica de la costa peruana. La empresa que representa a esta aplicación se llama Project Control AI, adjunto logotipo en la carpeta, integra en el aplicativo toda esta información, adicionalmente deseo que incluyas no solo área de acero sino también la cantidad de acero comercial (Ejemplo: 4 varillas de acero de Ø 5/8), detallar el tipo de acero para los estribos, y el diseño de los estribos.
info Instruye al LLM a programar una herramienta técnica de diseño de concreto armado con cálculo detallado de varillas comerciales y espaciamiento de estribos.
labelPrompt: Presupuesto S10 en Excel
Elabora un presupuesto sintético para un proyecto de carretera con aproximadamente 30 actividades, en formato S10, que incluya metrados, precios unitarios y montos, además de costos indirectos. En una pestaña adicional, incorpora los análisis de precios unitarios formulados con la cuadrilla, cantidad, rendimiento y totales, siguiendo el formato S10 y asegurando la trazabilidad entre los análisis de PU y el presupuesto. El resultado debe ser en Excel.
info Útil para solicitar planillas presupuestarias complejas con trazabilidad de costos unitarios y fórmulas activas en Excel.
